Personal Finance Manager

---Brief description---
A console app to help track your income and spending

---How to use---
The console app will first prompt you for your name, age, and current balance. Then, a list of available options will appear:

1- View Balance
2- Graph Spending
3- Add Money
4- Buy
5- Exit program

The user can choose any option by typing its number and then hitting "enter".

---Options functionality---
View Balance: Shows the current balance
Graph Spending: Shows where the money went by graphing the amount spent for each category
Add Money: Increases your balance
Buy: Records a purchase
Exit program: Terminates the app

---Concepts Applied---
OOP: Classes, Methods, Fields, and Properties
Basic programming: Variables, flow control (for loop, while loop, goto), conditions (if statements, switch statement)
